# Multi-Select Event Registration

## Overview

Users can select multiple events from the calendar and register for all of them at once.

## How to Use

### 1. Enable Multi-Select Mode
- Click the **"Multi-Select"** button in the calendar view (must be signed in)
- Button turns blue, purple banner appears with instructions

### 2. Select Events
- Click events in the calendar to select them
- Selected events show a **âœ“ checkmark** and blue highlight
- Click again to deselect

### 3. Register for All
- Click **"Register for Selected Events"** button
- Fill in registration details once
- Click **"Register for X Events"**
- All registrations are processed together

### 4. Exit Multi-Select Mode
- Click **"Cancel Multi-Select"** or complete registration

## Features

- **Visual Feedback**: Checkmarks and blue rings on selected events
- **Single Form**: Fill details once for all events
- **Waitlist Support**: Full events are added as waitlist requests
- **Batch Processing**: All registrations submitted together

## Notes

- Only available in Calendar view (not List view)
- Must be signed in to use multi-select
- Weekly registration limits still apply
